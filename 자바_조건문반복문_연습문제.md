# 별찍기

다음 내용을 출력해보기

```
*
**
***
****
```


```java
package exercise;

public class Exercise05 {

	public static void main(String[] args) {
		for (int i = 1 ; i < 5 ; i++) {
			for (int j = 1; j <= i; j++) {
				System.out.print("*"); 
				// println은 출력 후 개행. print는 개행 X 
				if (i == j) {
					System.out.println();
				}
			}
		}
	}
}
```
