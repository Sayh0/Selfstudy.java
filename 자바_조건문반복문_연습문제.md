# 별찍기

다음 내용을 출력해보기

```
*
**
***
****
```


```java
package exercise;

public class Exercise05 {

	public static void main(String[] args) {
		for (int i = 1 ; i < 5 ; i++) {
			for (int j = 1; j <= i; j++) {
				System.out.print("*"); 
				// println은 출력 후 개행. print는 개행 X 
				if (i == j) {
					System.out.println();
				// i == j 해지면 개행 하라는 뜻. 그럼 다음 줄 별 찍힘.  
				}
			}
		}
	}
	
}

```

역순으로 출력하기

```
****
 ***
  **
   *
```

```java
package exercise;

public class Exercise06 {

	public static void main(String[] args) {
		for (int i = 1; i < 5; i++) {
			for (int j = 4; j > 0; j--) {
			if (i < j) {
				System.out.print(" ");
				} else {
					System.out.print("*");
					}
				}
			}
			System.out.println(); // 이렇게 하면 개행이 한번밖에 안됨
			}
		}

	
```

## 십자가 별 찍기
홀수 m, n을 입력받아 다음과 같은 패턴을 출력하는 코드를 작성하시오. 아래 예시는 m = 5,n = 11을 입력했을 때의 결과이다.
```
     +
     +
+++++++++++
     +
     +
```
1. 세로줄은 패턴의 정 가운데에 위치해야 한다.   
2. 만약 입력된 m과 n 중 하나라도 짝수인 경우 "홀수를 입력하세요"라는 오류 메세지를 출력하고, 패턴은 출력하지 않는다.

```java
package hsquiz;
import java.util.Scanner;
public class hsquiz_cross {
	public static void main(String[] args) {
		Scanner sc = new Scanner(System.in);
		System.out.println("홀수 두 개를 입력하세요(공백으로 구분)");
		int n = sc.nextInt();
		int m = sc.nextInt();
		int i, j, k;
		int upperstar = (m-1)/2;
		
		if ( m % 2 != 0 && n % 2 != 0 ) {
			for (i = 0;i < upperstar; i++) {
				for (k = 0; k < (n-1)/2; k++) { // 수정
					System.out.print(" ");
				}
				System.out.println("*");
			}
			for (j = 1;j < n; j++) {
				System.out.print("*");
			}
			System.out.println("*");
			for (i = 0;i < upperstar; i++) {
				for (k = 0; k < (n-1)/2; k++) {
					System.out.print(" ");
				}
				System.out.println("*");
			}
		}
		
		else {
			System.out.println("홀수를 입력하세요");
			 }
	}
}
```
