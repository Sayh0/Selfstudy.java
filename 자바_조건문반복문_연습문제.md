# 별찍기

다음 내용을 출력해보기

```
*
**
***
****
```


```java
package exercise;

public class Exercise05 {

	public static void main(String[] args) {
		for (int i = 1 ; i < 5 ; i++) {
			for (int j = 1; j <= i; j++) {
				System.out.print("*"); 
				// println은 출력 후 개행. print는 개행 X 
				if (i == j) {
					System.out.println();
				// i == j 해지면 개행 하라는 뜻. 그럼 다음 줄 별 찍힘.  
				}
			}
		}
	}
	
}

```

역순으로 출력하기

```
****
 ***
  **
   *
```

```java
package exercise;

public class Exercise06 {

	public static void main(String[] args) {
		for (int i = 1; i < 5; i++) {
			for (int j = 4; j > 0; j--) {
			if (i < j) {
				System.out.print(" ");
				} else {
					System.out.print("*");
					}
				}
			}
			System.out.println(); // 이렇게 하면 개행이 한번밖에 안됨
			}
		}

	
```
이렇게 하면 이상하게 나옴. 오답.
맞는 코드는 이렇다.

```java
package sec01.exam01;

public class Exercise06 {

	public static void main(String[] args) {
		for (int i = 1; i < 5; i++) { 
			for (int j = 4; j > 0; j--) { //for2
				if (i < j) {
					System.out.print(" ");
				} else {
					System.out.print("*");
				}
			}
			System.out.println(); // 개행까지 반복되게 for문 안으로.
		}
	}
}
```
